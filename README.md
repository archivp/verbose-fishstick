# Kekwcirclegame Season 2

## Запуск

```bash
npm install
copy .env.example .env
npm start
```

Открой: http://localhost:3000

Если порт 3000 занят, поменяй `PORT=3001` в файле `.env`.

## Тестовый админ

Логин: `admin`
Пароль: `admin123`

## Страницы

- `/` — главная
- `/login.html` — вход
- `/register.html` — регистрация
- `/cabinet.html` — кабинет игрока
- `/games.html` — таблица игр и очков
- `/admin.html` — админ-панель
- `/admin-games.html` — редактирование игр
- `/admin-users.html` — редактирование игроков

Данные хранятся в `data/database.sqlite`.
Аватары загружаются в папку `uploads/`.


Изменения этой версии:
- страница колеса удалена;
- пункт «Колесо» удалён из меню;
- на странице игр удалена колонка «Действие»;
- отметка прохождения осталась только в личном кабинете в пуле игр профиля.


## Список игр и очки
Добавлен список игр из сообщения пользователя. Очки выставлены по простой системе: 1 час среднего прохождения = 1 очко. Значения приблизительные и их можно менять в админке.

## Обновление дизайна
В этой версии интерфейс переведён на минималистичный стиль: светлый фон, простые карточки, округлые кнопки, чистые таблицы и без неоновых эффектов.
