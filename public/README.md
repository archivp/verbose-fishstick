# Kekwcirclegame — Season 2

Готовый мини-сайт игрового события с аккаунтами, авторизацией, личным кабинетом, таблицей участников и админ-панелью.

## Запуск

```bash
npm install
npm start
```

Откройте: http://localhost:3000

## Тестовый админ

- Логин: `admin`
- Пароль: `admin123`

## Что внутри

- Главная страница с баннером события
- Регистрация пользователей
- Вход в аккаунт
- Личный кабинет игрока
- Профиль с ником, аватаром и статистикой
- Таблица участников
- Админ-панель для управления игроками
- Тёмная игровая стилистика с анимациями

## Важно

Для настоящего продакшена поменяйте `JWT_SECRET` в переменных окружения.
